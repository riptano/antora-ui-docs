== Get private endpoints information with the DevOps API

Use the DevOps API to create, remove, and manage private endpoints.

You can use the DevOps API to perform the actions your xref:manage:org/manage-permissions.adoc[role permissions] allow.

include::ROOT:partial$endpoint_perm.adoc[]

=== Prerequisites

Ensure you have xref:manage:org/manage-permissions.adoc[permission] to manage private endpoints.

[NOTE]
====
To increase your security, consider xref:manage:devops/devops-api.adoc#devops-access-list[restricting public access] to your database using the access list.
====
