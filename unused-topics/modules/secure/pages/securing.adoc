= Securing {astra_db}
:page-tag: security,admin,astra-db

Because security is a priority for all modern cloud applications, {company} has implemented several protocols to ensure security remains a priority.

{company} {astra_db} is a SOC2 Type2-compliant database that is secured by default including encryption over the wire and at rest.

For details on {astra_db} security, check out the link:https://www.datastax.com/resources/whitepaper/astra-security[{astra_db} Security Whitepaper].

The following sections discuss security policies that are specific to {astra_db}.
To view security protocols implemented across {company}, see link:https://www.datastax.com/products/datastax-enterprise/security-assurance[Security assurance, window="_blank"] on the {company} website.
